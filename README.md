Customer Segmentation | K-Means Clustering Machine Learning Project

Developed an end-to-end Machine Learning clustering project to segment customers into distinct groups by applying the complete ML workflow, from data collection and preprocessing to model development, visualization, and business insight generation.

The project utilized the Mall Customers dataset from Kaggle. It involved data cleaning, preprocessing, feature selection, and exploratory data analysis using Annual Income and Spending Score to identify customer segments through unsupervised machine learning.

Implemented the K-Means Clustering algorithm and used Within-Cluster Sum of Squares (WCSS) with the Elbow Method to determine the optimal number of clusters, resulting in five meaningful customer segments. The analysis uncovered valuable business insights by identifying high-value customers, moderate spenders, and low-spending customer groups, enabling personalized marketing campaigns, targeted offers, discount strategies, and improved customer retention to maximize business profitability.
